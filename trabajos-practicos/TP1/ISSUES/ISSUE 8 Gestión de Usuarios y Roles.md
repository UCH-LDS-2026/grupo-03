## Gestión de usuarios y roles

**User Story:**  
Como administrador quiero gestionar usuarios y roles para controlar el acceso al sistema.

**Detalle:**  
- Crear, editar y desactivar usuarios  
- Asignar roles: usuario, técnico, administrador  
- Asociar usuarios a un área  

**Criterios de aceptación:**  
- Se pueden crear y modificar usuarios correctamente  
- Cada usuario tiene un rol asignado  
- Los usuarios pueden ser activados o desactivados
